# 4 天最小落地优先级

如果时间很紧，按这个顺序做：

```text
1. web/index.html + WebSocket 事件流
2. NPC 主动行为 activity_worker
3. NPC 之间互动 interaction_worker
4. story_event 简单模板
5. mood/current_goal 字段
6. relationship 可先只写 event_logs
```

不要一开始把 `emotion`、`relationship`、`story` 都做复杂。MVP 可以先用规则和模板驱动。
