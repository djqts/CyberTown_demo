### 2.3 NPC 移动全链路

```mermaid
sequenceDiagram
    participant S as Scheduler
    participant RMQ as RabbitMQ
    participant EW as EventWorker
    participant NW as NPCWorker
    participant DB as PostgreSQL
    participant BW as BroadcastWorker
    participant WS as WebSocket Hub
    participant G as Godot WsClient
    participant TM as TownMap

    S->>RMQ: Publish town.tick {day, minute}
    RMQ->>EW: Consume town_tick_event

    EW->>NW: HandleTownTick(tick)
    NW->>DB: FindActiveMoves(townID, minute)
    DB-->>NW: []NPCMove

    loop 每个移动
        NW->>NW: HandleMoveRequested(move)
        NW->>DB: MoveNPC(npcID, newLocationID)
        NW->>RMQ: Publish npc.moved {npc_id, from, to}
    end

    NW->>RMQ: Publish npc.idle.action (20%概率)

    RMQ->>BW: Consume town_broadcast (npc.moved)
    BW->>WS: Broadcast {type:"npc.moved", data:{...ids...}}
    WS-->>G: JSON via WebSocket

    G->>G: WsClient._process() poll
    G->>G: TownStore.handle_ws_message()
    G->>G: _handle_npc_moved(data)
    G->>TM: update_npc() → npc_updated signal
    TM->>TM: _on_npc_updated()
    TM->>TM: BFS path → move_along_path()
```

